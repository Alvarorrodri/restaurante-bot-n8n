# 🍽️ Bot de Atención al Cliente con IA para Restaurantes

Bot de Telegram con memoria conversacional que gestiona 
la atención al cliente de restaurantes automáticamente usando IA.

## ¿Qué hace?

- Responde preguntas sobre menú, horarios y ubicación
- Gestiona reservas de mesa paso a paso
- Recuerda el contexto de toda la conversación
- Personalizable para cualquier restaurante en minutos

## ¿Cómo funciona?

1. El cliente escribe al bot por Telegram
2. El AI Agent procesa el mensaje con toda la información del restaurante
3. La memoria conversacional mantiene el contexto entre mensajes
4. El bot responde de forma natural y gestiona la reserva completa

## Tecnologías usadas

- n8n (automatización)
- Telegram Bot API
- Groq API (Llama 3.3 70B)
- Simple Memory (memoria conversacional)

## Cómo usarlo

1. Importa el JSON en tu instancia de n8n
2. Conecta tus credenciales de Telegram y Groq
3. Modifica el prompt con los datos de tu restaurante
4. Activa el flujo y listo

## 💰 Casos de uso comerciales

Este bot es replicable para cualquier negocio local:
- Restaurantes y bares
- Peluquerías y centros de estética
- Clínicas y consultas médicas
- Tiendas y comercios locales
